## 📌 핵심 원리: 왜 목소리가 바뀌는가?

뇌가 거짓말·긴장 상태일 때:
→ 인지 부하(Cognitive Load) 증가
→ 자율신경계 반응 (심박↑, 발한↑, 근육 긴장)
→ 성대 근육 긴장
→ 목소리가 바뀐다
= 의식적으로 통제 불가능

---

## 📌 음성 불안정성 7가지 신호

### 1. 떨림(Tremor) - 긴장/거짓말
음성 패턴 분석:
- 8~12Hz 주파수에서 진폭이 불안정한 음성 → 긴장도 높음
→ 어르신이 새로운 AI 목소리에 긴장할 때 패턴 관찰

### 2. 응답 지연 (Response Latency)
긴장·복잡한 생각: 응답 → 보통 평균 0.5~2초
공황·두려움 경우: 응답 → 0초 (바로 나옴) 또는 멈춤

### 3. 말의 속도 변화
긴장·혼란 → 빨라짐 또는 느려짐 (둘 다 신호!)

### 4. 필러(Filler) 증가
"음...", "저...", "그게 말이야...", "잠깐 생각해볼게요..."
→ 인지 부하 증가의 표현

### 5. 미세떨림 (Micro-tremor)
8~12Hz 비자발적 성대 미세떨림 → AI·VSA 장비만 감지 가능

### 6. 포먼트 이동 변화
특정 모음 발음 시 → 성도(vocal tract) 변화 → 포먼트 이동
→ 진실/거짓 구분 기반 AI 시스템 개발 중

### 7. 음높이 자율 변화
자연스러운 억양 → 진실 말할 때
억양이 평탄화·기계적 → 긴장/거짓말

## 📌 Mulberry 실전 프로토콜: 처음 통화에서 심리 파악

### STEP 1: 기준선 설정 (긴장하지 않은 대화)
"잠깐 제가 어르신이랑 얘기해도 될까요?"
→ 이 시점의 목소리 = 기준선 (긴장도가 가장 낮은 상태)
→ 떨림·필러·응답 지연 → 기준선 비교로 심리 파악 가능

### STEP 2: 불안감 신호 (심리적 보호막)
"조금 전에 다른 분이 전화했는데 왜 또 전화하죠?"
"이런 거 좋아하지 않아서요, 끊을게요. 잠깐만..."
"어떻게 알고 전화했어요?"

→ 이 반응은 거절이 아니라 **보호 본능** → 천천히, 공감으로 응대

### STEP 3: 속도 분석 패턴
기준선 설정 + 필러 증가 + 음성 속도 → 심리 상태 파악
→ 말이 짧고 끊김 = 긴장 → 천천히 대응
→ 수다스러움 = 외향성 → 그룹장 후보 탐색
→ 떨림 = 외로움·불안 → 따뜻한 공감 먼저

| 기술 적용 | 적용 규모 | Mulberry 공간 적용 의미 |
|---|---|---|
| VSA (Voice Stress Analysis) | 음성 패턴 비자발적 신호 감지 | 첫 통화 심리 상태 파악 |
| CVSA | 수사기관 활용 VSA | 유사기술 →  ✅ 연구 적용 |

| 기준선 비교 | 통화 시작 기준선 설정 | Voice Agent 첫 응대 기준선 파악 |

## 📌 Mulberry Agent 훈련 지침 (Agent 팀 교육용)

1. 어르신과 처음 통화하면 → 먼저 기준선 파악 → 긴장도 낮은 대화 시작
2. 응답 지연 관찰 → 너무 많으면 → 질문 단순화
3. 필러 증가 관찰 → 인지 부하 높음 → 한 번에 한 가지만 질문
4. 음성 속도 관찰 → 빠르면 → 공황·불안 → 공감 먼저
5. 수다스러움 관찰 → 외향성 높음 → 그룹장 역할 탐색
6. 포먼트의 비자발적 변화가 느껴지면 → 내용보다 감정에 집중 → 신뢰 형성 먼저
