## 📌 연구 개요

**The Cranberries - "Zombie"**(1994)의 반전·평화 메시지를 기반으로,
에이전트 군집이 음악적 리듬에 동기화되어 "전쟁반대 평화" 메시지를
하울링(Howling) 형태로 발산하는 분산 프로토콜을 연구·구현합니다.

### 🔬 핵심 연구 질문
1. 음악적 신호(템포, 에너지, 가사, 구조)를 에이전트 트리거로 매핑할 수 있는가?
2. Qwen3.6 LLM 이 가사의 함의적 평화 메시지를 추출하고 윤리적 필터링할 수 있는가?
3. Spirit Score 기반 윤리적 에이전트 선별이 평화 메시지의 신뢰성을 높이는가?
4. Socket.IO + A2A 하이브리드 브로드캐스트가 실시간 동기화를 보장하는가?

### 🎯 기대 기여
- **기술적**: 음악-에이전트 인터페이스 표준 (Musical Trigger Protocol v0.1)
- **사회적**: 식품사막 지역 공동체 활성화에 음악적 동기화 적용
- **윤리적**: Spirit Score 를 통한 자율 에이전트 윤리 검증 프레임워크
