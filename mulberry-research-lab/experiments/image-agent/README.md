# 🖼️ Mulberry Image Agent

이미지 파일에서 텍스트를 추출하고, 그 내용을 해석하여 Mulberry AI 에이전트 명령어로 변환하는 실험 프로젝트.

## ✨ 현재 기능 (MVP)
- 이미지 업로드 → OCR로 텍스트 추출
- 규칙 기반 키워드 매칭으로 기본 명령어 생성
- 명확한 명령어가 없을 경우 사용자에게 질문하여 명령어 보완
- (시뮬레이션) Mulberry Agent 호출

## 🚀 실행 방법 (Colab)
1. `image_agent.py` 코드를 Colab에 복사
2. 이미지 업로드 셀 실행
3. 분석 및 질문에 응답
