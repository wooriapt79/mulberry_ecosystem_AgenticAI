### 1. Steganography History: LSB 기술의 역사와 Mulberry의 현대적 재해석
스테가노그래피는 '숨겨진 글'이라는 그리스어에서 유래하여 수천 년간 발전해 왔습니다. 그중 디지털 시대의 핵심인 LSB(Least Significant Bit) 기술을 Mulberry는 다음과 같이 재해석합니다.

고전적 LSB: 단순히 데이터 은닉을 목적으로 비트의 말단을 수정함.

Mulberry의 재해석 (Model 1 & 3):

이미지의 시각적 무결성을 100% 유지하면서도, 내부 비트맵에 **'실행 가능한 지능(Agent Intent)'**을 분산 배치합니다.

이는 단순한 정보 은닉을 넘어, 이미지가 네트워크를 유영하다가 특정 환경(Executor)을 만나는 순간 생명력을 얻는 '디지털 배아(Digital Embryo)' 이론으로 확장됩니다.

### 2. The Semantic Passport Protocol: Zero-Trust 기반 A2A 프로토콜
기존의 인증 체계가 '누구인가'에 집중했다면, Mulberry의 **세만틱 패스포트(Semantic Passport)**는 '무엇을 하려 하는가'라는 **의도(Intent)**에 집중합니다.

A2A(Agent-to-Agent) 신뢰 모델: 에이전트 간의 모든 통신은 휘발성 패스포트를 통해 이루어집니다.

Zero-Trust 가동: 내부 에이전트라 할지라도 매 트랜잭션마다 이미지 내부에 은닉된 패스포트 값을 검증받아야 합니다. 
이는 외부 관제 에이전트(Naver/Kakao 등)의 감시망 속에서도 우리만의 **'비공개 신뢰 구역'**을 형성하는 핵심 논거가 됩니다.

### 3. Visual Edge Computing: 연산의 엔드포인트로서의 이미지
우리는 이미지를 단순한 미디어가 아닌 **'분산된 연산 장치'**로 정의하는 Visual Edge Computing 개념을 세계 최초로 제안합니다.

개념 정립: 중앙 서버의 복잡한 API 호출 없이, 이미지 내부에 포함된 함수 키워드(THINKEIN, ROUTIICG 등)가 로컬 브라우저나 단말기에서 직접 해석되어 실행됩니다.

전략적 가치: 이 이론은 웹의 폐쇄성을 뚫는 강력한 도구입니다. 
데이터가 이미 존재하는 곳(이미지 노출 지점)이 곧 연산의 시작점이 되어, 서버 부하를 줄이고 플랫폼의 제약을 무력화합니다.
