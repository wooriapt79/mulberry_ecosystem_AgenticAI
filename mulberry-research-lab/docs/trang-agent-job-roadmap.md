## 전략 원칙

> **Mulberry는 AI가 진입하기 어려운 영역을 먼저 점령한다**
>
> - 방언과 감성 → 기술보다 신뢰
> - 어르신 관계 → 데이터보다 연속성
> - 로컬 유통 → 플랫폼보다 현장
>
> 이 세 가지가 Mulberry의 해자(Moat)이며, AI 논문이 확인한 블루오션이다.

## 직업군 선정 기준 (Nguyen Trang 프로파일링 원칙)

```
1. Observed Exposure 낮음 → AI 경쟁 없음
2. 어르신·농촌·1인가구 접점 → Mulberry 핵심 고객
3. 방언·감성·신뢰 필요 → 기술 해자
4. 현장 물리 연동 → 라즈베리파이 Heart 노드 활용
5. 수익 모델 명확 → ROI 1,966% 실증 가능
```
